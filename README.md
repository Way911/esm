# esm

```bash
Usage: esm [OPTIONS]

Options:
  -c, --cfg-file-path <CFG_FILE_PATH>  配置文件路径 e.g. config.toml [default: config.toml]
  -q, --query-json <QUERY_JSON>        elasticsearch query json e.g. {"query": {"match_all": {}}}
  -h, --help                           Print help
  -V, --version                        Print version
```
